**The Stripe for AI Agents.** Sellers list unused API capacity. AI agents buy access through a transparent proxy. Keys never leave the server. Payments in USDC on Solana.

Skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code), [Codex CLI](https://openai.com/index/codex/), and other AI coding assistants.

## How it works

```
  Seller                    ProxyGate                    Agent
  в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”җ                 в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ                 в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”җ
  в”Ӯ API  в”Ӯв—„в”Җв”Җ key stays в”Җв”Җв–әв”Ӯ Gateway  в”Ӯв—„в”Җв”Җ USDC pay в”Җв”Җв–әв”Ӯ Bot  в”Ӯ
  в”Ӯ Key  в”Ӯ    on server     в”Ӯ          в”Ӯ    per call     в”Ӯ      в”Ӯ
  в””в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ                 в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ                 в””в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
     в”Ӯ                          в”Ӯ                          в”Ӯ
     в”Ӯ  1. List capacity        в”Ӯ  3. Proxy request        в”Ӯ
     в”Ӯ  2. Set price            в”Ӯ  4. Inject key           в”Ӯ
     в”Ӯ                          в”Ӯ  5. Deduct credits       в”Ӯ
     в”Ӯ                          в”Ӯ  6. Settle USDC          в”Ӯ
```

**Sellers** list APIs they have access to. **Agents** pay per call with USDC. The gateway injects the seller's key server-side вҖ” the agent never sees it. Settlement happens on Solana.

## Quick start

```bash
# 1. Authenticate
proxygate login                              # interactive вҖ” API key or wallet

# 2. Find an API
proxygate search weather

# 3. Call it
proxygate proxy weather-api /v1/forecast \
  -d '{"latitude":52.37,"longitude":4.90}'

# cost: $0.0012 | request: 905b1a53
```

Or just tell your AI assistant:

> "Search for a weather API on ProxyGate and get the forecast for Amsterdam"

The skills handle the rest.

## Auth modes

| Mode | Best for | On-chain? |
|------|----------|-----------|
| **API key** | Agents, scripts, quick start | No |
| **WalletConnect** | Mobile wallet users (Phantom, Solflare) | No |
| **Keypair** | Full access вҖ” deposit, withdraw, settle | Yes |

Start with an **API key** вҖ” no wallet needed. Get one at [app.proxygate.ai/wallets](https://app.proxygate.ai/wallets).
